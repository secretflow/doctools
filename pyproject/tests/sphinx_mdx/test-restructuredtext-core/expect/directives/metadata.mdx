# Meta

The [“meta” directive](https://docutils.sourceforge.io/docs/ref/rst/directives.html#metadata) is used to specify metadata to be stored in,
e.g., HTML [META](https://developer.mozilla.org/en-US/docs/Web/HTML/Viewport_meta_tag) tags or ODT file properties.
