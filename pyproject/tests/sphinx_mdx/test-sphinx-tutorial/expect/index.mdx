# Welcome to Lumache's documentation!

<strong>Lumache</strong> (/lu'make/) is a Python library for cooks and food lovers that
creates recipes mixing random ingredients.  It pulls data from the [Open Food
Facts database](https://world.openfoodfacts.org/) and offers a <em>simple</em> and
<em>intuitive</em> API.

Check out the [Usage](usage.mdx) section for further information, including how to
[install](usage.mdx#installation) the project.

<Container type="note">
  This project is under active development.
</Container>
