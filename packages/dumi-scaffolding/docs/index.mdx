# Werkzeug

[![](_static/werkzeug-horizontal.png)](https://werkzeug.palletsprojects.com)

<em>werkzeug</em> German noun: “tool”.
Etymology: <em>werk</em> (“work”), <em>zeug</em> (“stuff”)

Werkzeug is a comprehensive [WSGI](https://wsgi.readthedocs.io/en/latest/) web application library. It began as
a simple collection of various utilities for WSGI applications and has
become one of the most advanced WSGI utility libraries.

Werkzeug doesn’t enforce any dependencies. It is up to the developer to
choose a template engine, database adapter, and even how to handle
requests.
